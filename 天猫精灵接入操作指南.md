### 第三步：创建服务文件

创建 `server.py`：

```python
from flask import Flask, request, jsonify
import aiohttp

app = Flask(__name__)

# ===== 配置 =====
HA_URL = "http://localhost:8123"  # HomeAssistant 地址
HA_TOKEN = "你的HomeAssistant令牌"  # 下面教你怎么获取

# 设备映射（根据你的实际情况修改）
DEVICE_MAP = {
    "灯": {"domain": "light", "entity": "light.living_room"},
    "客厅灯": {"domain": "light", "entity": "light.living_room"},
    "卧室灯": {"domain": "light", "entity": "light.bedroom"},
    "空调": {"domain": "climate", "entity": "climate.ac"},
    "窗帘": {"domain": "cover", "entity": "cover.curtain"},
}

@app.route('/webhook', methods=['POST'])
def webhook():
    """天猫精灵回调接口"""
    
    try:
        data = request.json
        print("收到请求:", data)
        
        # 解析用户说的话
        utterance = data.get('utterance', '')
        
        # 简单命令解析
        result = process_command(utterance)
        
        return jsonify({
            "returnCode": "0",
            "returnMessage": "success",
            "result": {
                "type": "TTS",
                "text": result
            }
        })
        
    except Exception as e:
        print(f"错误: {e}")
        return jsonify({
            "returnCode": "1",
            "returnMessage": str(e)
        })

def process_command(text: str) -> str:
    """处理命令"""
    text = text.lower()
    
    # 开关灯
    if "开" in text and "灯" in text:
        return call_ha("light.living_room", "turn_on")
    if "关" in text and "灯" in text:
        return call_ha("light.living_room", "turn_off")
    
    # 空调
    if "开" in text and "空调" in text:
        return call_ha("climate.ac", "turn_on")
    if "关" in text and "空调" in text:
        return call_ha("climate.ac", "turn_off")
    
    # 查询温度
    if "温度" in text or "多少度" in text:
        return get_temperature()
    
    return "我不知道该怎么做"

def call_ha(entity_id: str, service: str) -> str:
    """调用 HomeAssistant API"""
    try:
        import requests
        domain = entity_id.split('.')[0]
        url = f"{HA_URL}/api/services/{domain}/{service}"
        
        r = requests.post(
            url,
            json={"entity_id": entity_id},
            headers={"Authorization": f"Bearer {HA_TOKEN}"}
        )
        
        if r.status_code == 200:
            return "好的，已执行"
        else:
            return f"执行失败：{r.text}"
    except Exception as e:
        return f"错误：{str(e)}"

def get_temperature() -> str:
    """获取温度"""
    try:
        import requests
        url = f"{HA_URL}/api/states/climate.ac"
        
        r = requests.get(
            url,
            headers={"Authorization": f"Bearer {HA_TOKEN}"}
        )
        
        if r.status_code == 200:
            data = r.json()
            temp = data.get("attributes", {}).get("current_temperature")
            return f"当前温度 {temp} 度"
        return "无法获取温度"
    except Exception as e:
        return f"错误：{str(e)}"

if __name__ == '__main__':
    print("🚀 服务启动：http://localhost:5000")
    print("Webhook 地址：需要公网访问")
    app.run(host='0.0.0.0', port=5000)
```

### 第四步：安装 HomeAssistant（可选但推荐）

如果不用 HA，需要直接控制天猫精灵支持的设备，会比较复杂。

先用 Docker 安装 HA：

```powershell
# 安装 Docker Desktop (https://www.docker.com/products/docker-desktop)

# 运行 HomeAssistant
docker run -d \
  --name homeassistant \
  --network=host \
  -v ./ha-config:/config \
  homeassistant/home-assistant:stable
```

打开浏览器访问 http://localhost:8123

### 第五步：获取 HomeAssistant Token

1. 打开 http://localhost:8123
2. 点击左下角用户名
3.往下拉 → 创建令牌
4. 复制令牌，替换到上面的 `HA_TOKEN`

### 第六步：让服务能被外网访问

有两种方式：

#### 方式 A：使用内网穿透（FRP）

```bash
# 下载 FRP
wget https://github.com/fatedier/frp/releases/download/v0.52.3/frp_0.52.3_linux_amd64.tar.gz

# 配置 frpc.ini
[common]
server_addr = 你的VPS地址
server_port = 7000

[web]
type = tcp
local_ip = 127.0.0.1
local_port = 5000
remote_port = 5000
```

#### 方式 B：使用 Cloudflare Tunnel（免费推荐）

```bash
# 安装 cloudflared
wget -q https://github.com/cloudflare/cloudflared/releases/latest/download/cloudflared-linux-amd64 -O /usr/local/bin/cloudflared
chmod +x /usr/local/bin/cloudflared

# 登录
cloudflared tunnel login

# 创建隧道
cloudflared tunnel create lobster-webhook

# 运行
cloudflared tunnel run --url http://localhost:5000 lobster-webhook
```

会给你一个公网 URL，比如：`https://xxx.trycloudflare.com`

## 🎯 简化的替代方案

如果你觉得太复杂，可以先用这个简单架构：

```
天猫精灵 App → 智能设备（天猫精灵支持的）
                    ↓
            通过 HA 联动 → 龙虾军团 Agent
```

### 具体操作

1. **在 HA 中添加天猫精灵设备**
   - 购买天猫精灵支持的智能设备（灯泡、插座等）
   - 在天猫精灵 App 中添加
   - 在 HA 中通过"阿里巴巴飞燕"集成添加

2. **配置自动化**
   - HA 中创建自动化
   - 当设备状态变化时，调用 Agent

## 📞 如果遇到问题

| 问题 | 解决方法 |
|------|----------|
| Python 没安装好 | 重新安装，勾选 Add to PATH |

| 外网访问不了 | 使用 Cloudflare Tunnel |
| 天猫精灵不响应 | 检查 Webhook 地址是否正确 |
